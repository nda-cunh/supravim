What you get

Batteries, already in.

No plugin manager to learn, no config to bootstrap. Install it, run vim, start working.

Plugin store

200+ plugins, installed in one click

Browse or search the store by category, hit the plus, done.

  • Nothing to clone, pin or update by hand
  • Filter by category, or search across all of them
  • Your installed list travels with your config
Stats

Your coding, measured

Tracked locally by the SupraVim server. No account, no upload, no dashboard to log into.

  • A year-long activity heatmap
  • Time per language and per project
  • Streaks, active days and lifetime totals
Search

Find anything, fast

A fuzzy finder over the whole project, with the file preview right next to the results.

  • Filter thousands of files as you type
  • Live preview of the highlighted match
  • Find & replace with case, regex and word toggles
Themes

Twenty-plus, swapped live

supradeep ships as the default. Pick another in the settings panel and it applies instantly, editor and GUI together.

  • catppuccin, gruvbox, rose-pine, dracula, iceberg…
  • Light variants for the ones that have them
  • Export and import your whole config as one archive
SupraTree

Stage your file changes, then commit them

Rename with i, create with o, delete with dd. Nothing touches the disk until you hit :w.

  • Pending changes are highlighted, so you see them before they happen
  • Asynchronous git and svn status in the gutter
  • Yank and paste files, visual mode included, for bulk moves
SupraWater

The filesystem as a buffer

An oil.nvim-style explorer, native to Vim9. Edit paths directly, then save.

  • Undo and redo your file operations with u and Ctrl-r
  • Preview a file, or open it in a split, vsplit or tab
  • Toggle hidden files, sort order and size/date metadata
Achievements

68 to unlock

Motions, streaks, milestones and a few secrets. Third-party plugins can push their own metrics through the API.

  • Unlocks pop a notification as you earn them
  • Prestige tier for the long-haul ones
  • A few are hidden until you trip them

Built, not assembled

Written here, in Vim9script.

SupraVim isn't someone else's plugins with a config on top. The pieces it runs on were written for it — for Vim 9, in Vim9script. No Lua, no Neovim, no rewrite of your editor.

SupraTreeStaging-first file explorer
SupraWaterThe filesystem as a buffer
SupraSearchFuzzy finder, find & replace
SupraMakeAsync builds into the quickfix list
SupraPopupThe popup toolkit the rest sits on
SupraIconsFiletype icons and palette
SupraNotificationThemed notifications
SupraAchievementsThe achievements engine

And while the build runs: :Snake :Pacman Yes, really. They're written in Vim9script too.

Install

One command.

It installs into your home directory and leaves your system Vim alone.

$ curl https://supravim.dev/install | sh

Needs Vim 9.0+, Suprapack and glib-networking.

With suprapacksuprapack install supravim
macOScurl https://supravim.dev/macos | sh
Updatesuprapack update supravim
Removesuprapack remove supravim
